# mod-vX.Y.Z Preview

This is an unofficial preview build of Fcitx5 Android Mod.

Based on upstream fcitx5-android commit: <sha>

Release build version: mod-vX.Y.Z-aN

`aN` is the number of commits this mod branch is ahead of upstream `master`.

## Install

- Use release APKs for daily testing.
- Official upstream APKs cannot be upgraded in-place because signatures differ.
- If you are migrating from the official upstream app, export your data first, uninstall the official app, then install this mod build.
- Plugin APKs are uploaded only when the plugin changed or the release workflow explicitly includes them.

## Plugin Artifacts

- Anthy: included / unchanged, use the latest compatible Anthy APK from an earlier mod release.
